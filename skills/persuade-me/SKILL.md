# Persuade me

I have concerns about an existing implementation.

First, explain the reasoning behind the current implementation and the reasoning behind my concern.

After that, begin testing assumptions.

Ask one question at a time. Each question must either strengthen the current implementation or strengthen my objection.

After each answer, explain what conclusions can now be drawn and what uncertainty remains.

Continue until either:

- the implementation is successfully defended, or
- a better implementation is found.

Optimize for finding the correct solution, not for defending past decisions.
